# AXT-Logo

綾急技研のロゴ画像です。  
色違い差分と拡張子差分がいくつかあります。

## 差分

「`背景色`-`本体色`.`拡張子`」で統一しています。

> alphaが透明、whiteが純白`#FFFFFF`、blackが純黒`#000000`  
> AXTblueが技研コーポレートカラー("AXT-blue", `#222244`)  
> AXTwhiteが技研サブコーポレートカラー("AXT-white", `#DDDDEE`)
